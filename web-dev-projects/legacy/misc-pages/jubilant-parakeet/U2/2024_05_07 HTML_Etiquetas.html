Modelo de contenido en HTML

En HTML cada elemento tiene su propio modelo de contenido, el cual describe el tipo de contenido que debe contener.

Categorias

  • Elementos para seccionar.
  • Elementos para incrustar
  • Elementos de metadata.
  • Interactivos.
  • Fraseo de títulos.

Algunos de estos elementos pueden pertenecer a dos o más categorías por su función y definición.

Nuevos elementos HTML5

Con la llegada de HTML5 surgieron nuevos elementos, unos estructurales y otros semánticos.

Los nuevos elementos estructurales son:

  • <header>,
  • <section>,
  • <article>,
  • <aside>,
  • <footer>,
  • <nav> y
  • <main>.

Los elementos semánticos de HTML5 son los elementos de contenido. Estos son:

  • <figure>,
  • <figcaption>,
  • <video>,
  • <audio>,
  • <canvas>,
  • <data>,
  • <mark>,
  • <time>,
  • <details>,
  • <summary> y
  • <progress>.
