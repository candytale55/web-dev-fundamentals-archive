Elemento o Etiqueta <div>

<div> es un elemento o etiqueta de HTML. No es un elemento semántico, pero no ha desaparecido de HTML5 y <div> todavía tiene un uso perfectamente válido. Sirve para unir elementos cuando no puedes asociar significado semántico a un conjunto.

Es muy habitual usar <div> para envolver el contenido de una página y luego usar CSS3 para aplicarle estilos.

Al no ser un elemento semántico, debe evitarse cuando puede sustituirse por una etiqueta semántica. Existen etiquetas nuevas para organizar mejor una web que tenga muchas etiquetas <div>.

En los casos en que haya una necesidad de agrupar, se debe evitar el uso abusivo de las etiquetas <div> y sí usar las nuevas etiquetas que concretan más sus funciones. Pero no todas las etiquetas que antes se llamaban <div> ahora pasan a llamarse <section>.
