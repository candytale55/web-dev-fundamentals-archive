DOCTYPE

La etiqueta DOCTYPE, situada al principio del documento HTML, se utiliza para especificar a los navegadores el lenguaje y versión utilizada.

La sintaxis es:

<!DOCTYPE [Elemento de primer nivel] [Disponibilidad] “[+/-] // [Organización]// [Tipo] [Nombre]//[Idioma]” “[URL]”>
  • [Elemento de primer nivel]-elemento del nivel superior declarado en el DTD.
  • [Disponibilidad]-se indica la disponibilidad del identificador. Puede ser:
    • Public-identificador público.
    • System-identificador de sistemas.
    • URL-identificador local o URL.
  • [+/-]-símbolo que depende de la organización. En el caso de W3C el signo es negativo.
  • [Organización]-organización la cual ha creado el DTD que se va a utilizar
  • [Tipo]-tipo de documento que se está incluyendo, DTD.
  • [Nombre]-nombre del DTD
  • [Idioma]-idioma en el que está escrito el DTD
  • [URL]-La URL del DTD.

¿Qué es el DTD que se utiliza en el DOCTYPE?

El DTD es la definición del tipo de documento.

El DOCTYPE es la forma de declarar el tipo de documento.

El DTD es dónde se declara el DOCTYPE, esto es, dónde se define la estructura que debe tener el documento. Por lo tanto, se utiliza DOCTYPE para informar qué DTD se está usando.

La W3C ya tiene creados los diferentes DTD publicados y se utilizan a modo de estándar tanto para desarrolladores como por los navegadores.

La información de la etiqueta DOCTYPE también es leída por los validadores cuya función es comprobar que los elementos utilizados cumplen con el estándar indicado.

Declaración DOCTYPE para HTML5

En HTML5 el DOCTYPE, cadena más corta y simple, que le corresponde es el siguiente:

<!DOCTYPE html>

Los motores de búsqueda también saben, sólo por esta declaración, el lenguaje de marcado utilizado y evidentemente, premian con mejores posiciones aquellas que se hayan desarrollado con HTML5 porque indica que la página está actualizada.

Prueba a buscar en el código fuente de diferentes páginas web y averigua si están desarrolladas en HTML5.
