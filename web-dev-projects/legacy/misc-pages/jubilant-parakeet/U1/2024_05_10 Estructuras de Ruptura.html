Estructuras de Ruptura en JavaScript

Son estructuras de control. En JavaScript se pueden tomar decisiones utilizando tres tipos de estructuras:

  1. If
  2. If...else
  3. Switch

Una sentencia condicional es una instrucción en la que se hace una comparación y según el resultado verdadero o falso (true or false) de la misma, el programa seguirá ejecutando una u otra instrucción.

La estructura de decisión más simple que se puede tomar es la condicional.

    If

    Es un condicional que sirve para realizar unas u otras operaciones en función de una expresión.

    La estructura If ejecuta u omite una serie de sentencias dependiendo de si la comprobación da verdadero o falso.

    La sintaxis es:

    
if (expresión que debe cumplirse) {
   acción que se realizará
  }

    Solo si se cumple la expresión la acción se realizará.

    
if (“llueve”) { 
  alert(“me quedo en casa”);
}

    Nota: Las llaves que incluyen la acción son opcionales, y lo mismo el sangrado. Si encontráis código que no los tenga, no quiere decir que sea incorrecto, pero es muy recomendable ponerlos.

    If else

    A diferencia de la estructura if donde sólo se tiene en cuenta si una expresión es cierta; en la estructura if else, se tiene en cuenta tanto si es cierta como si no lo es.

    Sintaxis:

    
if (expresión que debe cumplirse) {
 acción que se realizará
}
else {
Acción que se realizará en caso de que la anterior no se realice
}

    Ejemplo:

    
if( dia == “domingo”) { 
  document.write (“¡Feliz día!”);
  }
else { 
  Document.write (“¡Día de trabajo!”)
  }

    Si en este ejemplo quisiera poner un mensaje diferente para cada día de la semana, ni el If ni el If ... else me valen pero sí si utilizo la estructura switch.

    Switch

    Se utiliza esta estructura cuando se quieren poner acciones diferentes en función del resultado que sale.

    
switch (expresión) {
case 1:
  sentencia
break
case2:
  sentencia
break
.
.
.
default:
  sentencia en caso de que no se cumpla las anteriores
}

    break y default

    En las estructuras de decisión hay dos expresiones de ruptura:

    • break: La instrucción break pone fin al bloque y hace que el programa salte a la instrucción siguiente a la sentencia.
    • default: La instrucción default es opcional, su finalidad es que se ejecute algún código cuando ninguna de las condiciones se cumple.

    Ejemplo con switch

    
Switch (día de la semana) {
case 1:
  document.write (“¡Es lúnes. Iniciamos la semana!”)
break
case 2:
  document.write (“¡Es Martes!”)
break
case 3:
  document.write (“¡Es miércoles. Medio de la semana!”)
break
case 4:
  document.write (“¡Es jueves!”)
break
case 5:
  document.write (“¡Por fín es viernes!”)
break
case 6:
  document.write (“¡Es fin de semana!”)
break
case 7:
  document.write (“¡Es fin de semana!”)
break
default:
  document.write (“Ese día no existe”)
}

    Dependiendo del día de la semana se va a ejecutar una acción diferente. Cada día es un caso diferente. Se utiliza break (romper) para cambiar de caso (es opcional, pero es mejor). Y por si acaso no escribió bien el día de la semana está la expresión por defecto default para los demás casos (default es opcional también).
