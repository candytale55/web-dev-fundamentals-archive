Caracteres de Control

JavaScript define un mecanismo para incluir de forma sencilla caracteres especiales y problemáticos dentro de una cadena de texto, son los caracteres de control.

Se dice que se utiliza un mecanismo de escape. El mecanismo consiste en sustituir el carácter problemático por una combinación simple de caracteres.

  • \n Salto de línea
  • \t Tabulación
  • \’ Comilla simple
  • \” Comillas dobles
  • \\ Barra inclinada

Objetos

Javascript posee algunos objetos predefinidos: Array, Boolean, Date, Function, Global, Math, Number, Object, RegExp, y String.

Adicionalmente se pueden crear objetos nuevos, con sus propios métodos y propiedades.

También se podrían decir que son tipos de datos: Null (nulo) o Undefined (dato no definido).

Operadores Lógicos

AND lógico: símbolo &&

El resultado será true si todas las comparaciones son verdaderas.

  • True&&true = true
  • True&&false= false
  • False&&true= false

OR lógico: símbolo ||

El resultado será true solamente con que se cumpla una de las comparaciones.

  • True || false =true
  • False|| true = true

Operador No (!)

Su función es la de invertir el valor que se le pasa, es decir, lo que era verdadero es falso y lo que era falso es verdadero.

Iguales: símbolo == / No iguales: símbolo !=

Compara dos valores y devuelve true si ambos son iguales.

Idénticos: símbolo ===

Compara dos valores y también el tipo de datos y devuelve true si ambos son idénticos en valor y en tipo de dato, por lo tanto si uno fuera un valor numérico y otro un carácter daría como resultado false.

No idénticos: símbolo !==. Invierte el sentido de la comparación.
